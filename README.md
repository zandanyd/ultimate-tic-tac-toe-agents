# ultimate-tic-tac-toe-agents
Ultimate Tic-Tac-Toe (UTTT) game environment and adversarial AI agents (Minimax with Transposition Tables, and rollout-truncated Heuristic MCTS). Built in Python with Pygame for human evaluation play.
